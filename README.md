# vue-form

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### adding nightwatch to existing vue project
```
vue add e2e-nightwatch
```

### to run the nightwatch
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
